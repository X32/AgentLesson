# Chapter07 项目架构与模块功能文档

## 项目概述

**项目名称**: AI 视频课程处理系统  
**技术栈**: FastAPI + SQLModel + MySQL + 阿里云 Dashscope  
**核心功能**: 视频上传→AI 处理→在线学习→在线考试→智能问答

## 二、工程文件模块功能详解

### 2.1 核心代码模块

#### 📄 main.py - Web 应用入口 (8.6KB)

**API 路由功能**:

| 路由 | 方法 | 功能描述 |
|------|------|----------|
| `/` | GET | 首页渲染，显示视频列表 |
| `/detail/{videoid}` | GET | 视频详情页 (思维导图/摘要/关键帧) |
| `/login` | POST | 用户登录，生成 AES-Token |
| `/checklogin` | POST | Token 有效性验证 |
| `/upload` | POST | 视频上传 + AI 全流程处理 |
| `/exam/{videoid}` | GET | 考试页面渲染 |
| `/exam/submit/{videoid}/{token}` | POST | 试卷提交 + 自动评分 |
| `/stream` | POST | 流式 AI 问答 (SSE) |
| `/index` | GET | 索引页面 |

---

#### 📄 module.py - 业务逻辑核心 (20.8KB)

**认证模块**:
- `aes_encrypt(source)` - AES-CBC 加密登录信息生成 Token
- `aes_decrypt(encrypted)` - Token 解密获取用户信息
- `check_token(token)` - 验证 Token 有效期和权限

**音视频处理模块**:
- `video_audio(filename)` - 视频转音频 (moviepy), 读取时长
- `audio_text(filename)` - 语音识别 (Whisper-large-v3-turbo)
- `gen_key_frames(filepath)` - 关键帧提取 (OpenCV 帧差异对比)

**AI 内容生成模块 (Dashscope/qwen-plus)**:
- `summarize(text)` - 生成课程摘要/笔记
- `gen_xmind_json(text)` - 生成思维导图 JSON 结构
- `write_xmind_file(xmind_json, filename)` - 写入.xmind 文件
- `gen_exam(text)` - 生成考题 (8 选择 +3 简答)
- `insert_exam(examtext, videoid)` - 考题入库
- `gen_reference(keyword)` - 阿里云搜索参考链接
- `ai_score(user_answer, userid, examid, question, ai_answer)` - AI 批改简答题

**数据持久化模块**:
- `insert_score(userid, videoid, examid, answer, score)` - 考试成绩入库

---

#### 📄 model.py - 数据模型定义 (1.7KB)

**SQLModel ORM 模型**:

| 模型 | 表名 | 主要字段 |
|------|------|----------|
| `Users` | users | userid, username, password, realname, role, createtime |
| `Videos` | videos | videoid, videotitle, filename, duration, content, summary, keyword, reference, xmindjson, examjson, createtime |
| `Exams` | exams | examid, videoid(FK), question, type, answer, score, options, createtime |
| `Scores` | scores | scoreid, userid(FK), videoid(FK), examid(FK), answer, score, createtime |

**数据库连接**: `mysql+pymysql://root@localhost:3306/llm_backend`

---

#### 📄 schedule.py - 后台定时任务 (2.5KB)

**功能**: 每 60 秒轮询一次数据库，检查并补充缺失的 AI 处理结果

```
轮询检查流程:
├─ 检查 content 为空 → 调用 audio_text() 语音识别
├─ 检查 summary 为空 → 调用 summarize() 生成摘要
├─ 检查 xmindjson 为空 → 调用 gen_xmind_json() + write_xmind_file()
├─ 检查 reference 为空 → 调用 gen_reference() 搜索参考
├─ 检查 examjson 为空 → 调用 gen_exam() + insert_exam()
└─ 检查关键帧目录不存在 → 调用 gen_key_frames() 生成
```

### 6.3 各阶段处理时间和依赖

| 阶段 | 处理时间 | 外部依赖 | 可并行 |
|------|----------|----------|--------|
| 视频上传 | 取决于文件大小 | 无 | 否 |
| 音视频转换 | 5-30 秒 | moviepy | 否 |
| 语音识别 | 30 秒 -5 分钟 | faster_whisper | 否 (GPU 加速) |
| AI 摘要 | 3-10 秒 | 阿里云 Dashscope | 是 |
| AI 思维导图 | 5-15 秒 | 阿里云 Dashscope | 是 |
| AI 考题 | 5-15 秒 | 阿里云 Dashscope | 是 |
| 参考搜索 | 1-3 秒 | 阿里云 OpenSearch | 是 |
| 关键帧提取 | 10-60 秒 | OpenCV | 是 (可异步) |
| 数据入库 | <1 秒 | MySQL | 否 |

**总处理时间**: 1-5 分钟 (取决于视频时长和硬件配置)
